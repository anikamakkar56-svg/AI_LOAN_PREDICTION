# 🏦 Intelligent Loan Approval Prediction

An AI-powered Loan Approval Prediction System built using **Machine Learning**, **Scikit-learn**, and **Streamlit**. This application predicts whether a loan application is likely to be approved based on applicant details such as income, education, employment status, credit history, and property area.

---

## 📌 Project Overview

The Intelligent Loan Approval Prediction System helps financial institutions automate the loan approval process by analyzing applicant information using a trained Machine Learning model. The application provides real-time predictions through an interactive and user-friendly Streamlit interface.

---

## ✨ Features

- 🤖 Machine Learning-based Loan Approval Prediction
- 📊 Real-time Prediction
- 🧹 Data Preprocessing & Missing Value Handling
- 🔄 Categorical Data Encoding
- 🌐 Interactive Streamlit Web Application

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Intelligent-Loan-Approval-Prediction.git
```

Go to the project folder:

```bash
cd Intelligent-Loan-Approval-Prediction
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
